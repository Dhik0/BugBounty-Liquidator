## 探测漏洞类型 (Target Vulnerabilities)
# 本工具专注于探测以下由于配置疏忽导致的 高危信息泄露与接口暴露：
1. 环境变量泄露 (Environment Snippets):
探测 /.env、/.git/config、/.vscode/sftp.json。
价值：获取数据库密码 (DB_PASSWORD)、API 密钥及云服务凭证。
2. Spring Boot 运行监控泄露 (Actuator Endpoints):
探测 /actuator/env、/heapdump、/jolokia。
价值：获取内存中的敏感配置、连接池信息及系统变量。
3. CMS 接口暴露 (CMS Interfaces):
探测 /xmlrpc.php (WordPress)、/phpinfo.php。
价值：确认是否存在暴力破解接口或服务器环境敏感信息泄露。
4. 源代码管理遗留 (Source Code Leaks):
探测 /.git/、/.svn/ 等目录。
价值：通过泄露的源码进行离线审计，发现深层业务逻辑漏洞。

## 快速开始
1. 将目标域名放入 `rpc_scanner.py` 文件83行
2. 运行脚本：`rpc_scanner.py`此脚本可自动获取该域名的所有子域名
3. 注意：请将非扫描项目添加至exclude.txt文件中
4. 结果将自动提炼至 `out`目录下
5. 运行脚本：`extract_targets.py`此脚本可自动从`out`目录下的文件中提取至`vulnerable_found.txt`文件中
